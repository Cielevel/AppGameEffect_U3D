# Universal controller 说明

## 目的

1. 建立通用的角色控制系统，包括但不限于：
   1. Behavior--Movement
   2. Behavior--Attack
   3. Behavior--Interaction
   4. Control--Animation
2. 建立通用的按键输入系统（通过 new Input System）
   1. 支持键盘、手柄、触摸屏等输入设备
   2. 输入按键可自定义

## 分类

概念上可能与 Utility 混淆，但目前在 AppGameEffect 中进行开发，后续更新到 Utility 工程
